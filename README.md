# Rotaract Club Treasury Management System

## Overview
This project is a web-based application developed to digitize the traditional pen-and-paper treasury system of a Rotaract club. It provides a centralized platform to manage financial records, documents, and requests efficiently.

## Features
- Upload and manage financial documents (receipts, cheques, requests)
- Centralized treasury record system
- Easy access to all financial data in one place
- User-friendly interface for smooth navigation
- Digital tracking of transactions and records

## Tech Stack
- **Frontend:** HTML5, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
